Sicherheit
==========

Die Absicherung von Domain-Objekten ist in Mapbender über ein Berechtigungssystem implementiert.
Es erlaubt globale Berechtigungen auf Bereiche und individuelle Berechtigungen auf Objekte.
Eine flexible Rechtezuweisung ist in diesen Bereichen möglich:

Rechteverwaltung
****************

Mapbender bietet zur Absicherung der Anwendungsbereiche unterschiedliche Stufen von Berechtigungen an, die Sie vergeben können.

- **Erstellen**: Ein neues Objekt erzeugen
- **Ansehen**: Ein existierendes Objekt lesen
- **Bearbeiten**: Ein existierendes Objekt editieren
- **Löschen**: Ein existierendes Objekt entfernen
- **Berechtigungen verwalten**: Ermöglicht die Bearbeitung von Berechtigungen
- **Aktualisieren**: Ermöglicht die Aktualisierung von Datenquellen


Fügen Sie Benutzer und Gruppen über ``Sicherheit`` → ``Globale Berechtigungen`` → ``Rechteverwaltung`` diesem Bereich hinzu, damit sie Zugriff auf das Rechteverwaltungssystem erhalten.
